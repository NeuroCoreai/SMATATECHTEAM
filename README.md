# SMATATECHTEAM
Aplikasi Bendahara Cerdas adalah solusi manajemen kas kelas berbasis web, dirancang untuk meningkatkan transparansi dan efisiensi pencatatan keuangan di lingkungan sekolah.  Aplikasi ini dibangun menggunakan arsitektur Frontend (UI/UX modern dengan Tailwind CSS) dan Backend (Service Layer JavaScript), yang menjamin kode yang bersih dan profesional.
